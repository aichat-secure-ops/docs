The **Facebook Comment Automation** feature enables businesses to listen to comments made on published Facebook posts.  You can specify to listen for specify keywords or intents and provide a personalised comment and DM response direct to the Facebook user's inbox. Some use cases are:

- \*\*Lead gen campaigns: \*\* lead users to submit a lead gen form upon 'dm me' request
- \*\*WhatsApp Marketing Updates Opt-in: \*\*lead users to your WhatsApp number and opt-in into marketing updates

![Facebook Comment Automation Action](/images/facebook-comment-automation-action.gif)

# How to setup

1. **Create a comment automation**: In AiChat Dashboard, go to \*\*Flows â†’ Comment Automation.  \*\*Click **Create Comment Automation**.
1. Provide a campaign name that helps identify the nature of this automation.
1. Click into the newly created campaign and select the Facebook post you wish to monitor.  You can select more than 1 posts or select all
1. Configure when will trigger the automation
   1. **Comment Requirements**: this determines what Facebook users' comments will trigger the automated follow-up
   1. **Media Requirements**: this determins if Facebook users need to include image attachments along with the comments to trigger the automated follow-up
   1. **Invalid comment handling**: activate this if you need to handle specific scenarios where media was not included when needed, or user triggered with specific keyword outside of campaign duration.
1. In the 3rd step, you can define the followu-up response
   1. **Reply message**: this will be a comment responded to Facebook user's comment.
   1. **Send a private message (DM)**: You can setup a single message to be sent directly to the user's inbox along with an optional button.  
1. Review the automation along with the start and end date for automation.
